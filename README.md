## Fortnite Maps
### Distance and ETA calculator for Fortnite Battle Royale

Try it out: [maps.livfor.it](http://maps.livfor.it)

![Gif showcase](showcase.gif)

#### How to:
Click anywhere to spawn a new marker.
Click on an existing marker to move it.
